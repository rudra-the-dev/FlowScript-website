Reference

Every command,
explained simply

FlowScript has a small, powerful set of commands. Learn them all in under 5 minutes.

Output
say
Print text or a variable to the output.
say "Hello World"
say 'name'
Variables
create
Create a variable with an initial value. Use single quotes for variable names.
create a number called 'age' with value 20
set 'age' to 25
Math
add / subtract / multiply / divide
Perform math operations on variables.
add 5 to 'age'
subtract 2 from 'age'
multiply 'age' by 2
divide 'age' by 4
Conditions
if / elif / else
Run different code based on conditions. Always close with end.
if 'score' is greater than 90
    say "Excellent"
elif 'score' is greater than 70
    say "Good"
else:
    say "Try again"
end
Loops
repeat
Repeat an action a number of times.
repeat 5 times say "hello"
Input
ask
Ask the user a question and save their answer to a variable.
ask "What is your name?" and save to 'name'
say 'name'
Android Power

Automate anything
on your phone

FlowScript controls your Android phone using AI vision. Write what you want to happen in plain English — FlowScript figures out how to do it.

Messaging
Send WhatsApp messages
Open WhatsApp, find a contact by name, and send them a message automatically.
open "WhatsApp"
find contact "Mom"
send message "On my way home"
Scheduling
Run at a specific time
Schedule any automation to run at a set time every day. Runs in the background even when the app is closed.
every day at "8:00am"
    open "WhatsApp"
    find contact "Mom"
    send message "Good morning!"
Apps
Open and control any app
Open any installed app and interact with it using plain English commands.
open "YouTube"
search for "lo-fi beats"
tap first result

open "Chrome"
go to "google.com"
Triggers
React to phone events
Run automations automatically when something happens on your phone.
when battery is below 20
    notify me "Charge your phone"

when connected to wifi "Home"
    open "Spotify"
Notifications
Send yourself notifications
Push notifications to your phone from any FlowScript automation.
notify me "Meeting in 10 minutes"

every day at "9:00am"
    notify me "Time to drink water"
How it works
AI-powered vision
FlowScript takes a screenshot of your screen, sends it to an AI model, and the AI tells FlowScript exactly what to tap, type, or swipe. Works on any app, any phone.
Your code → FlowScript interpreter
→ Screenshot → AI vision model
→ tap X,Y → action performed
→ repeat until task done
Note
Automation commands require the FlowScript Android app installed on your phone. The playground above only runs basic FlowScript — open, send, notify and scheduling commands run on the app only.
